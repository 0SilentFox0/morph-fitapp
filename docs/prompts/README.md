# Prompts Log

Each user prompt is saved as a separate file. The filename format is `{id}_{short_description}.md`.

**Format:**
- `prompt_id`: Unique identifier (e.g. 001, 002)
- `user_prompt`: Summary or excerpt of the user's request
- `tasks_completed`: List of task IDs from TASKS.md that were completed during this prompt
- `date`: When the prompt was processed
